# FCM-practices
Fuzzy cognitive maps

To run this code you need to install FCMpy
### FCMpy requires python >=3.8.1 (*) and depends on:

    pandas>=1.0.3
    numpy>=numpy==1.18.2
    scikit-fuzzy>=0.4.2
    tqdm>=4.50.2
    openpyxl

FCMpy is available on PyPi! The latest version can be installed by:

    pip install fcmpy
